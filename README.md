# Автоматическое создание work-докладов
Программа автоматически формирует полноценный доклад на основе статьи из Википедии, используя предоставленную ссылку. Она не только извлекает и структурирует текстовую информацию, но и дополняет материал соответствующими иллюстрациями, делая содержание более наглядным и информативным. При этом оформление доклада выполняется в соответствии с установленными правилами и стандартами, что позволяет получить аккуратно оформленный, логически выстроенный и готовый к использованию документ.

## Как установить программу
Скачайте и разархивируйте код на своем компьютере.
Python3 должен быть уже установлен. Затем используйте pip(или pip3, если есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

## Как запустить программу
Чтобы запустить программу, надо в терменале написать
```
python main.py
```
и следовать дальнейшим инструкциям программы.

После ввода всех данных в программу в папке с программой появится файл `result.docx` с докладом и автоматически откроется
